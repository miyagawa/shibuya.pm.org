Shibuya Perl Mongers テクニカルトーク #5

Shibuya Perl Mongersテクニカルトーク#5 を2004年12月16日に開催します。

  • 日時 - 2004年12月16日 (木) 18:30-21:00 (18:00 開場)
  • 会場 - 神保町三井ビル 17F 株式会社インターネットイニシアティブ (IIJ) 大会議室
  • 料金 - 無料
  • 事前登録 - 定員を超えたので締め切りました。
  • プログラム 内容・順番・時間などは予告なく変更する場合があります。
    1. "iPod&iTunes Hacks" - 宮川達彦
    2. "再考ですかーっJavaScript" - 小山浩之
    3. "はてなの開発裏側見せますパート2" - 伊藤直也
    4. "PerlとJavaで異言語Webサービス連携 - SOAP/WSDLの光と影" - 竹迫良範
    5. "アイスホッケーの成績集計とテスト駆動開発" - 吉田裕美

プログラム詳細

iPod&iTunes Hacks
スピーカー: 宮川達彦
iTunes と iPod の API を利用した雑多なスクリプト、iTunes ファイル共有プロトコル DAAP の利用、AirMac Express のプロトコル解析などディープな Hack を紹介する。
再考ですかーっJavaScript
スピーカー: 小山浩之
どうもブラウザの都合で虐げられがちなJavaScript。プログラミング言語としてのJavaScriptを再考したうえで、Mozillaプロジェクトで利用されているJavaScriptの処理系SpiderMonkeyの利用方法や、その拡張方法を解説する。
はてなの開発裏側見せますパート2
スピーカー: 伊藤直也
最近のはてな、何をどんな風に開発しているか、その裏側をお見せします。はまぞう、はてな検索、新アンテナ、そして最近社内で熱いのは Google AdSense 代替広告 Hack。面白いとこどりで紹介させていただきます。
PerlとJavaで異言語Webサービス連携 - SOAP/WSDLの光と影 -
スピーカー: 竹迫良範
巷で流行のWebサービス、その中でもSOAPとWSDLの仕様は疎結合でプラットフォーム非依存の技術と言われています。しかし現実は…。PerlのSOAP::Liteモジュールを使って異言語Webサービス連携を実現するためのad hocを紹介します。
アイスホッケーの成績集計とテスト駆動開発
スピーカー: 吉田裕美
みなさんは、アイスホッケーのスコアーを見たことありますか？ この夏に開発したアジアリーグ・アイスホッケー成績集計システムでの複雑なルール等とそれを正しく実装する為のテスト駆動開発などに付いて、お話します。
JAPH を読む
スピーカー: 水野貴明
簡単に書けることをわざと複雑に書く。そんなパズルのようなスクリプトを読み解きながら、そこに使われている使われているテクニックを味わい、面白がり、よい意味での無駄な努力を賞賛します。
Perl大改造！！劇的ビフォーアフター
スピーカー: 早川真也
問題を抱えたプログラムを匠がリフォームを試みます。 Perlそのものをリフォームする荒技も登場！(patch無し)

スピーカー紹介

宮川達彦
Shibuya.pm リーダーにして、CPAN モジュール制作数で世界3本の指に入る。iTunes や Google, Skype などの API でちまちま遊ぶのがマイブームだが飽きるのも光速。著書に Blog Hacks など。
小山浩之
フリーのソフトウェアエンジニア兼ライター。最近ようやっと本来の姿であった「Perlの人」や「Apacheモジュールの人」的な仕事を請負えてきつつあるらしい。元来バイナリ操作やプロトコルハックを好むが最近は自制している。それでも、訳あって封印していたYahoo!メッセンジャーネタが緊急浮上中な三十路突入一児の父。著書"Apacheモジュールプログラミングガイド"。尊敬する人は及川 光博。
伊藤直也
近頃はカリスマブロガーやアルファギークなどの謎の業界横文字で呼ばれることも多い、はてなスタッフ。著書にBlog Hacks。株式会社はてな 取締役最高技術責任者。
竹迫良範
株式会社ドリーム・アーツ広島ラボ勤務、現在の職業はmod_perl屋さん。Namazu for Win32のメンテナ、Perlクックブック第2版の監訳などを担当。
吉田裕美
お昼ね好きなSOHOプログラマー、今年も PerlとJava のお仕事を半半くらいの比率でやってました。
水野貴明
コンピュータ系兼業ライター。近著に「ULTIMATE Perl」など。本職は出版系システム会社で働くシステムエンジニア。
早川真也
機能的で、大胆なコードを作り出す、“レキシカル変数の仕掛人”。 レキサとパーサを最大限に活かす事を常に心掛ける。

About Us

Shibuya Perl Mongers は東京地区とくに渋谷周辺のインターネット関連企業に勤務している Perl ユーザのコミュニティ形成を目指す非営利の団体です。主な活動内容はプログラミング言語 Perl に関係するメンバー主催の勉強会やインターネット上での啓蒙活動や情報交換です。Shibuya Perl Mongers は Perl を利用し、スキル向上を望む方であればどなたでも無料で参加できます。

What is Perl Mongers?

Perl Mongers(パールモンガース)とは、地域ごとのPerlユーザグループの集合体です。発祥はニューヨークで現在は世界各地に Perl Mongers が分布し、それぞれの地域でのコミュニティーを形成しています。
