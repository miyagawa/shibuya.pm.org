バッドノウハウカンファレンス 2004

Shibuya Perl Mongers Presents "バッドノウハウカンファレンス 2004" を2004年5月14日に開催します。

最新情報

  • 発表資料とレポートリンクのエントリを作成しました。
  • 平林純氏の招待講演が決定しました。 『史上最強科学のムダ知識』 を持参するといいかもしれません。

プログラム概要

  • 日時 - 2004年5月14日 金曜日 18:30〜21:00
    入場は 18時から受け付けています。
  • 会場 - 国立オリンピック記念青少年総合センター センター棟 403室
  • 料金 - 無料
  • 事前登録 - 満員御礼となりました。 誠に申し訳ありませんが会場の定員の都合のため、登録のない方の当日参加は受け付けておりません。どうぞご了承ください。
  • プログラム 内容・順番・時間などは予告なく変更する場合があります。
    1. 18:00～ 開場、受付、準備
    2. 18:30～ キーノート: "バッドノウハウとの付き合い方" (仮) - 高林哲
    3. 18:50～ "バッドノウハウと黒魔術"(仮) - 宮川達彦
    4. 19:20～ 招待講演 "オッパイ星人とバッドノウハウ(仮)" - 平林純
    5. 19:50～ 小休止
    6. 20:00～ Lightning Talks - 自慢のバッドノウハウ談義を披露しあおう
    7. 21:00～ 閉会、会場で懇親会 (22時までには解散)
  • 発表資料 - こちら

プログラム詳細

バッドノウハウとの付き合い方 (仮)
スピーカー: 高林哲
ついつい深追いしてしまうのはなぜか？ 深追イストとは？？ 効果的なバッドノウハウシェアリング、などなど。詳細未定。
バッドノウハウと黒魔術(仮)
スピーカー: 宮川達彦
バッドノウハウを生み出すソフトウェアに見られる共通項とは？エンジニアはバッドノウハウをどう吸収し、付き合っていくのか？
オッパイ星人とバッドノウハウ(仮)
スピーカー: 平林純
科学はまさにノウハウだ。しかも、とても役に立つノウハウだ。 しかし、それが「オッパイ星人の科学」ならどうだろう？ 「無駄科学」の作者が「効率的作業」について考えると…？

Lightning Talks Session

グーアマ効果　～失われたバッドノウハウ～
スピーカー: 山下達雄
「Googleの検索結果にAmazonアソシエイトIDが出る！」 アサマシドリブンなバッドノウハウ探求とその顛末。
日本Unix昔話
スピーカー: 野首貴嗣
概要: スピーカーの体験を元にUnixのバッドノウハウを現代的視点から再検証してみるテスト。
マクロの決死圏
スピーカー: 森本淳
とあるLinux関連団体で制作頒布していたドキュメントCDに関するバッドノウハウの思い出。
ユニックス・イズ・ザ・バッドノウハウ
スピーカー: 江渡浩一郎
世界で最も普及したバッドノウハウを、 その歴史と共にふりかえる。
TeX and only TeX
スピーカー: 新部裕
TeX でのプログラミングを語ろう。 TeX だけがプログラミングだ。 生き残るものにこそ価値がある。 20年, 不朽の BK.
Perl でどうよ?
スピーカー: 伊藤直也
C++開発者 Bjarne Stroustrup 氏は言った。 「Java はマーケティングの産物だよね」。 つまり、言語の種類は本質ではない。 では、それをいかにそれ伝えるか。 Perl Hacker 涙の直感的プレゼン手法。
「奥が深い症候群」の傾向と対策
スピーカー: 鈴木 聡
人はなぜ「奥が深い症候群」にはまり込むのか？ 「奥の深さ」から抜け出すには？ ひとつの見方を提示する予定．
Inno Setup STAND ALONE COMPLEX
スピーカー: 竹迫良範
某オープンソースソフトウェアのWindowsへのポーティング秘話。 世界最強のWindowsインストーラ作成ソフトInno Setupの手により、 数々のバッドノウハウが隠蔽され、一つの複合体が形成されていく様子を描く。
バッドノウハウのない世界 (仮)
スピーカー: 小山浩之
準備中。
深追い Perl (仮)
スピーカー: 早川真也
準備中。

スピーカー紹介

高林哲
バッドノウハウ研究家にして 一発ネタ系 がらくた ソフトウェアの専門家。今思えば Perl との出会いがバッドノウハウ研究の出発点となったような気がする。
宮川達彦
Shibuya.pm リーダーにして、CPAN モジュール生産数世界一を誇る。最近では日本の RSS の第一人者としても活躍中。著書（翻訳）に Perl データマンジング。株式会社ライブドア 執行役員 Chief Technology Architect。
平林純
（自称）技術系サイトhirax.netで駄文を垂れ流している。 「オッパイ星人の力学」を研究したせいか、人からは単なる「巨乳好き」だと思われているらしい。
山下達雄
某 IT メーカで何かについて研究している研究員。小さくて短いシンプル なプログラムを好む。学生時代は 茶筌や SUFARY の開発 に携わる。最近はダウンサイジングして、くっつき BBSを始めとし た「くっつきシリーズ」や chalow などの Web 系ツール を細々と作っている。
野首貴嗣
脱輪とガジェットのことなら俺にまかせろ！うへ。
森本淳
準備中
江渡浩一郎
自他ともに認める深追イスト。深追い禁止 = 深追い必至がモットー。
新部裕
ハノイの塔と low level プログラミングをこよなく愛する。 「普通のやつらの下を行け!」がモットー。
伊藤直也
某ISPで blog サービスの企画・開発を努めるエンジニア。Blog 検索エンジン FeedBack 開発者。Blog Hacks 執筆中だが日夜飲み歩き一向に進まない。
鈴木 聡
バッドノウハウをあさっての方向から観察する流浪の似非HCI研究者。
竹迫良範
Namazu for Win32のメンテナ。後継者探しに広島県から遠征。 現在の職業はmod_perl屋さん、株式会社ドリーム・アーツ広島ラボに勤務。
小山浩之
職場で「Webのフロントエンド書くのはもうお腹いっぱい」とのたまいつつ、趣味でApache moduleを書いていたりする矛盾の多い本格派駄目サラリーマン。著書にApacheモジュールプログラミングガイド。
早川真也
「CPANは"くぱん"と読む」を普及させようと目論むツワモノ。一日1つ新しい(マニアックな)オペレーションシステムをインストールすることを日課にしている。LL Saturday 2003 での活躍は記憶に新しい。株式会社ライブドア LindowsOS 開発チーム。

About Us

Shibuya Perl Mongers は東京地区とくに渋谷周辺のインターネット関連企業に勤務している Perl ユーザのコミュニティ形成を目指す非営利の団体です。主な活動内容はプログラミング言語 Perl に関係するメンバー主催の勉強会やインターネット上での啓蒙活動や情報交換です。Shibuya Perl Mongers は Perl を利用し、スキル向上を望む方であればどなたでも無料で参加できます。

What is Perl Mongers?

Perl Mongers(パールモンガース)とは、地域ごとのPerlユーザグループの集合体です。発祥はニューヨークで現在は世界各地に Perl Mongers が分布し、それぞれの地域でのコミュニティーを形成しています。
